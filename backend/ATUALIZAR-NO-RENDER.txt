ATUALIZAÇÃO DO SERVIDOR — CBOFERTAS BACKEND 5.2.0

1. Substitua a pasta backend do repositório por esta versão.
2. No Render, mantenha Root Directory como backend.
3. Build Command: npm install
4. Start Command: npm start
5. Health Check Path: /health
6. Opcional: configure MELI_ACCESS_TOKEN.
7. Abra /health e confirme version 5.2.0.
