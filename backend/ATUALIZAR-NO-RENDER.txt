ATUALIZAÇÃO DO SERVIDOR — CBOFERTAS BACKEND 4.1.0

1. Substitua a pasta backend do repositório por esta versão.
2. No Render, mantenha Root Directory como backend.
3. Build Command: npm install
4. Start Command: npm start
5. Health Check Path: /health
6. Opcional: configure MELI_ACCESS_TOKEN.
7. Abra /health e confirme version 4.1.0.

CORREÇÃO PRINCIPAL
- Valores brasileiros com ponto de milhares são preservados.
- Exemplo: 2.500 = R$ 2.500,00; 1.499,90 = R$ 1.499,90.
- O Radar deixa de exibir R$ 2,50 para produtos de R$ 2.500,00.
