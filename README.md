# PromoZap MELI V16

Projeto limpo com:
- Android WebView sem bibliotecas externas;
- backend Node.js para Render;
- consulta do Mercado Livre com API oficial + leitura da oferta visível;
- proxy de imagens;
- workflow GitHub Actions para gerar APK.

## Render
Root Directory: `backend`
Build Command: `npm install`
Start Command: `npm start`

## APK
Abra **Actions > Gerar APK V16** e baixe o artefato `PromoZap-MELI-V16-APK`.
