CBOFERTAS V5.0

CORREÇÕES IMPORTANTES
- Valores como 2.500 e 1.500 agora são tratados como R$ 2.500,00 e R$ 1.500,00.
- O Radar não transforma mais milhares em centavos.
- O backend deve mostrar version 5.0.0 em /health.

LINK DE AFILIADO
1. Busque ou escolha o produto no CbOfertas.
2. Toque em Gerar link de afiliado.
3. No Mercado Livre, toque em Compartilhar na faixa Ganhos.
4. Escolha CbOfertas na lista de aplicativos.
5. O link recebido será aplicado automaticamente à oferta.

IMPORTANTE
- O CbOfertas não inventa parâmetros de afiliado.
- O link é criado pelo próprio Mercado Livre na conta conectada ao aplicativo oficial.
- Mensagens, favoritos e histórico usarão o link recebido.

ATUALIZAR O RENDER
- Root Directory: backend
- Build Command: npm install
- Start Command: npm start
- Health Check Path: /health

GERAR O APK
- Envie o projeto completo para o GitHub.
- Execute Validar e Gerar APK CbOfertas V5.0.
- Baixe CbOfertas-V5.0-APK.
