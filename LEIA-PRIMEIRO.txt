PROMOZAP MELI — VERSÃO REORGANIZADA 15

Backend no Render:
- Root Directory: backend
- Build Command: npm install
- Start Command: npm start

O aplicativo separa, quando disponíveis:
- preço no Pix/destaque;
- preço em outros meios;
- preço anterior;
- vendedor;
- imagem por proxy do backend.

IMPORTANTE: preços do Mercado Livre podem mudar por variante, conta, região e forma de pagamento. Sempre confira os dados antes de publicar.

APK AUTOMÁTICO NO GITHUB:
Após enviar o projeto, abra a aba Ações > Gerar APK. Ao concluir, baixe o artefato PromoZap-APK.
